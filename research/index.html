cognitive POMDP modeling for sonifing graph exploration
with Danyang Fan, Saehui Hwang, Gordon Yeung
We investigate how BLV users explore sonified graphs, and use these behavioral patterns to train cognitive models that can inform future interface guidance strategies.

We developed a user study platform where participants interact with dynamically updating sonified graphs on an iPad for completing tasks. Researchers can fully monitor through Python backend that supports real-time graph updates, exploration recording, and data visualization.

Our modeling approach leverages anti-reinforcement learning principles, using data-driven fitting and hyperparameter tuning based on specific decision-making processes (e.g., Gaussian Process).

mARker 🌐
hybrid-interfaced spatial sketching via iPad AR, Apple Pencil, and an instantly crafted tracker
mARker is a hybrid-interfaced spatial-sketching tool that leverages the iPad and Apple Pencil Pro as a widely adopted sketching ecosystem, lowering implementation barriers and expands the interactive potentials.

We achieved stylus pose tracking with the iPad ecosystem's standardized hardware by adding only a self-fabricated tracking cube; meanwhile, we leveraged the new Apple Pencil Pro APIs and followed HIG to implement a rich set of gestures for sketching.

a handheld device with haptic guidance for accessible navigation
Tangible Compass 👋♿
*accepted to ACM Chinese CHI '24.
Aim to design a low-cost, open-source device providing additional sensory feedback to support accessible navigation, we introduce Tangible Compass, a handheld haptic device as an add-on for smartphone navigation functions, providing force guidance in various accessible navigation scenarios.

Its functionality is achieved through smartphone sensor data streaming, scenario-based algorithmic force rendering, and a compact one Degree of Freedom (1-DoF) kinesthetic device transmitting forces. We featured interaction scenarios like route orientating navigation and clearance detection.
